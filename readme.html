Сайт конфигуратор ПК, курсовая работа. Данный сайт предназначен для выбора комплектующих для сборки персонального компьютера, с наглядными сравнениями характеристик и тестами различных комплектующий, а также советами по выбору. Предполагалось создание базы данных комплектующих на хостинге, с последующим внедрением в интерфейс видеоплеера Youtube со ссылками на бенчмарки конкретных процессоров и видеокарт. Создание графика эффективности и соотношения производительности процессора к видеокарте, выбор оптимальной оперативной памяти под процессор, блока питания под платформу(процессор, материнская плата, видеокарта)
